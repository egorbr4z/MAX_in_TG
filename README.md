# MAX_in_TG
Программа на python, написанная на специальном GREEN-API для работы с HTTP API MAX. В начале программы присутствуют константы, при задании которых это должно начать работать. Данные для констант нужно брать с сайта Green API и бота, созданного через @BotFather. В GREEN-API вам надо создать developer аккаунт max, и привязать номер телефона с аккаунтом в MAX. Для примера, мы используем этого бота для получения сообщений из школьной группы, которая находится в Макс. Аккаунт для этого нужен лишь один. Бота, к которому вы привязали данный код, можно добавить в группу в телеграм, и он автоматически будет пересылать в неё полученные из группы в Макс сообщения.
