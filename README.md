# MAX_in_TG
Программа на python, написанная на специальном GREEN-API для работы с HTTP API MAX. В начале программы присутствуют константы, при задании которых это должно начать работать. Данные для констант нужно брать с сайта Green API и бота, созданного через @BotFather. В GREEN-API вам надо создать developer аккаунт max, и привязать номер телефона с аккаунтом в MAX. Для примера, мы используем этого бота для получения сообщений из шкошкольшкошкольной группы, которая находится в Макс. Аккаунт для этого нужен лишь один, остальным участникам регаться не надо будет.кошкольнколосоздаоз
